# Changelog

All notable changes to the ASP.NET Web Forms Plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-16

### Added
- Initial plugin release
- `aspnet-webforms-expert` agent với 15+ years expertise
- `aspnet-webforms-senior` skill cho comprehensive WebForms knowledge
- `ultra-think` skill cho 10-step deep analysis
- MCP server integration cho custom tools
- Comprehensive documentation (README, SETUP guide)
- Example configurations và troubleshooting guides

### Features
- Dynamic context loading từ CLAUDE.md/README.md
- Support cho N-tier, 3-layer, custom architectures
- Oracle và SQL Server integration patterns
- ViewState optimization strategies
- Security best practices (SQL injection prevention, XSS protection)
- Performance tuning guidelines
- Legacy code modernization paths
- AJAX patterns (UpdatePanel, ASHX handlers)

### Documentation
- Complete README với installation instructions
- Detailed SETUP guide cho Windows/Linux/Mac
- Testing procedures
- Troubleshooting section
- Contributing guidelines
